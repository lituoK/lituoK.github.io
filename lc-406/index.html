发表于 2020-11-16

题目

406. 根据身高重建队列

思路

排序+模拟

题目给定元素包含{h, k}两个关键字，然后进行排序，排序的规则是：

对于元素peoplei，它前面的i - 1个元素 peoplej(0 <= j < i),要有 peoplei.k 个元素满足 peoplej.h >= peoplei.h

算法

  1. 首先我们先把这些元素按照h升序排序，如果身高相同，则按照k升序排序；

  2. 建立一个由n个空位的数组。

  3. 我们遍历所有元素，将其插入到其应该插入的位置。

    • 假设所有人的身高都不相同， 对于元素i， 他的前面要有k个元素的身高是大于h的，所以我们要在元素i的前面留k个空位,然后在相应的位置上进行插入即可。

    • 如果有身高相同的元素, 如果有t个元素的身高是相同的，对于这t个元素中的第i个元素，因为我们元素i前面要有k个元素的身高是大于等于元素i的，所以我们需要把前i - 1个相同的元素也当作是空位，然后同样的，留k个空位，进行插入即可。
